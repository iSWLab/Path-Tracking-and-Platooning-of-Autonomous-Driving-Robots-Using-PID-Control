# PID 제어를 이용한 자율주행 로봇의 경로 추적 및 군집주행


## 프로젝트 소개
레고 마인드스톰 EV3 로봇을 활용한 자율주행 로봇의 경로 추적 및 군집주행 프로젝트입니다. PID 제어를 통해 경로를 추적하며, 초음파 센서를 사용하여 안전거리를 유지하고 군집주행을 수행합니다.
